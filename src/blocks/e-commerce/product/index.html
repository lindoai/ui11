Usage Guidelines

  • • Use high-quality product images to showcase items effectively
  • • Include star ratings and review counts for social proof
  • • Display clear pricing with original and sale prices when applicable
  • • Show stock status and availability information
  • • Provide quick action buttons for cart and wishlist
  • • Use badges for sales, promotions, or product status

Accessibility Features

  • • Descriptive alt text for product images
  • • Keyboard navigation for all interactive elements
  • • Screen reader announcements for price and rating information
  • • High contrast color schemes for better visibility
  • • Clear focus indicators on buttons and links
  • • Semantic HTML structure for assistive technologies
