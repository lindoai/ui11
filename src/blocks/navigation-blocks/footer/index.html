Usage Guidelines

Accessibility Features

  • • Semantic footer landmark for screen readers
  • • Keyboard navigation support for all links
  • • High contrast focus indicators
  • • Descriptive link text and proper ARIA labels
  • • Mobile-friendly touch targets and responsive design
