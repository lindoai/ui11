Usage Guidelines

  • • Use image carousels for showcasing visual content and galleries
  • • Implement testimonial carousels to build trust and social proof
  • • Feature product carousels on home pages and category sections
  • • Include navigation arrows and dots for better user control

Accessibility Features

  • • Keyboard navigation support with arrow keys
  • • ARIA labels for carousel controls and current slide
  • • High contrast focus indicators for interactive elements
  • • Auto-play pause on hover and focus for better accessibility
