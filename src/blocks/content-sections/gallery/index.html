Usage Guidelines

  • • Use high-quality images with consistent aspect ratios
  • • Include filter functionality for large galleries
  • • Implement lightbox for full-screen image viewing
  • • Add loading states for better user experience
  • • Consider lazy loading for performance optimization

Accessibility Features

  • • Alt text for all gallery images
  • • Keyboard navigation for lightbox functionality
  • • Focus management when opening/closing modals
  • • Screen reader announcements for filter changes
  • • Escape key support for closing lightbox
