Usage Guidelines

  • • Use basic lists for simple navigation or menu items
  • • Add avatars when displaying user or people-related content
  • • Include actions for interactive list items that need user operations
  • • Implement nesting for hierarchical data structures
  • • Maintain consistent spacing and alignment throughout
  • • Use hover states to indicate interactive elements

Accessibility Features

  • • Semantic list structure with ul/ol and li elements
  • • Keyboard navigation support for interactive lists
  • • ARIA labels for complex list items and actions
  • • High contrast focus indicators for keyboard users
  • • Screen reader announcements for list updates
  • • Proper heading hierarchy for nested lists
