Usage Guidelines

  • • Use collapsible sidebars for main navigation in desktop applications
  • • Provide clear visual feedback for active/current page
  • • Include proper accessibility attributes for screen readers
  • • Consider overlay variant for mobile devices

Accessibility Features

  • • Keyboard navigation support with tab and arrow keys
  • • Screen reader compatible with proper ARIA labels
  • • High contrast focus indicators for keyboard users
  • • Smooth transitions with reduced motion preference respect
