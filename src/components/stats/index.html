Usage Guidelines

  • • Use counters for impressive numbers that warrant emphasis
  • • Display metrics with meaningful context and change indicators
  • • Show trends with appropriate visual cues (arrows, colors)
  • • Use comparisons to highlight relationships between data points
  • • Ensure all statistics are accurate and up-to-date
  • • Provide clear labels and units for all numerical values

Accessibility Features

  • • ARIA labels for all statistical values and indicators
  • • High contrast colors for trend indicators
  • • Screen reader friendly descriptions for visual elements
  • • Keyboard navigation support for interactive statistics
  • • Clear color coding with additional text indicators
  • • Proper semantic HTML structure for data presentation
