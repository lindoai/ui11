Usage Guidelines

  • • Use appropriate avatar sizes for different contexts (large for headers, small for lists)
  • • Include status indicators to show user availability and online presence
  • • Provide clear contact information and social links for user engagement
  • • Display relevant statistics and progress indicators for gamification
  • • Keep social links organized and easily accessible with proper icons
  • • Use consistent formatting for user information across all profile types

Accessibility Features

  • • Alt text for all avatar images and profile photos
  • • High contrast ratios for text and background colors
  • • Keyboard navigation support for all interactive elements
  • • Screen reader friendly structure with proper heading hierarchy
  • • ARIA labels for status indicators and progress bars
  • • Focus indicators for social links and buttons
