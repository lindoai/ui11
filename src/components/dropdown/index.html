Usage Guidelines

  • • Use select-like dropdowns for simple option selection
  • • Add search functionality when there are many options (8+)
  • • Implement multi-select for selecting multiple related items
  • • Include clear visual feedback for selected state
  • • Ensure proper keyboard navigation and accessibility
  • • Keep option text concise and descriptive

Accessibility Features

  • • ARIA attributes for screen reader compatibility
  • • Keyboard navigation with arrow keys and Enter
  • • Focus management and proper tab sequence
  • • High contrast focus indicators
  • • Escape key to close dropdown
  • • Proper labeling and role attributes
