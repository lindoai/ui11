Usage Guidelines

  • • Use spinners for quick operations (under 3 seconds)
  • • Use skeleton screens for content-heavy loading states
  • • Match loading indicator size to the content being loaded
  • • Provide context with loading messages when helpful
  • • Consider using progress bars for longer operations

Accessibility Features

  • • ARIA live regions announce loading state changes
  • • Respects user's reduced motion preferences
  • • Clear visual indicators with sufficient contrast
  • • Screen reader friendly loading messages
  • • Keyboard navigation remains functional during loading
