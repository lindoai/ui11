Usage Guidelines

  • • Use image avatars when user photos are available for personal touch
  • • Fall back to initials when images aren't available or fail to load
  • • Use icon avatars for anonymous users or placeholder states
  • • Choose appropriate sizes based on context (small for lists, large for profiles)
  • • Include status badges to show user availability and presence

Accessibility Features

  • • Alt text on images provides context for screen readers
  • • High contrast borders improve visibility across backgrounds
  • • Color-coded status badges with semantic meaning
  • • Consistent sizing system for predictable layout
  • • Proper contrast ratios in both light and dark modes
