Usage Guidelines

  • • Use horizontal dividers to separate content sections vertically
  • • Use vertical dividers to separate content side-by-side
  • • Use text dividers to provide context for new sections
  • • Choose appropriate thickness based on the importance of separation
  • • Maintain consistent spacing around dividers for visual balance

Accessibility Features

  • • Semantic HTML structure with proper heading hierarchy
  • • High contrast borders that work in both light and dark modes
  • • Text dividers use proper background colors for readability
  • • Role="separator" for programmatic identification
  • • Clear visual distinction without relying solely on color
