Usage Guidelines

  • • Use flex-row for horizontal layouts and flex-col for vertical layouts
  • • Apply flex-wrap when you want items to wrap to new lines
  • • Use align utilities (items-*) for cross-axis alignment
  • • Use justify utilities for main-axis distribution
  • • Combine flex utilities for complex responsive layouts
  • • Consider flex-grow and flex-shrink for item sizing control

Accessibility Features

  • • Flexbox maintains semantic document order for screen readers
  • • Visual order changes don't affect tab navigation sequence
  • • Proper heading hierarchy preserved regardless of visual layout
  • • Focus indicators remain visible with flex layouts
  • • Responsive flex layouts support keyboard navigation
  • • Screen reader compatibility maintained across all flex utilities
