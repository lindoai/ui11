Label

Form labels for input fields with support for required indicators, optional text, and helpful descriptions to guide user input.

Must be at least 8 characters

Usage Guidelines

  • • Always use clear, descriptive labels that explain what input is expected
  • • Mark required fields with a red asterisk (*) for visual clarity
  • • Use help text to provide context, examples, or formatting requirements
  • • Keep label text concise while being descriptive enough for understanding
  • • Position help text consistently either above or below the input field

Accessibility Features

  • • Proper label-input association using for/id attributes
  • • ARIA labels for required field indicators
  • • aria-describedby linking help text to form controls
  • • Screen reader friendly required and optional indicators
  • • High contrast colors that meet WCAG accessibility standards
