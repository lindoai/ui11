Trigger Types

Different interaction methods to show and hide tooltips.

Hover to see this tooltip
Click triggered tooltip
Focus to see this tooltip 
<!-- Hover Trigger -->
<div class="relative group">
  <div class="absolute bottom-full left-1/2 transform -translate-x-1/2 mb-2 opacity-0 group-hover:opacity-100 transition-opacity duration-200">
    <div class="bg-neutral-900 dark:bg-neutral-700 text-white text-sm px-3 py-2 rounded-lg shadow-lg whitespace-nowrap">

    </div>
  </div>

  <div id="click-tooltip" class="absolute bottom-full left-1/2 transform -translate-x-1/2 mb-2 opacity-0 transition-opacity duration-200">
    <div class="bg-neutral-900 dark:bg-neutral-700 text-white text-sm px-3 py-2 rounded-lg shadow-lg">
      Click triggered tooltip
    </div>
  </div>
  <button id="click-trigger" class="px-4 py-2 bg-green-500 text-white rounded-lg">Click Trigger</button>
</div>

Usage Guidelines

  • • Keep tooltip content concise and informative (ideally under 100 characters)
  • • Use tooltips for supplementary information, not essential content
  • • Position tooltips to avoid covering important UI elements
  • • Use semantic colors to reinforce the nature of the information
  • • Ensure tooltips are accessible via keyboard navigation

Accessibility Features

  • • High contrast colors meet WCAG accessibility standards
  • • Focus-visible triggers for keyboard navigation
  • • Screen reader compatible with proper ARIA attributes
  • • Appropriate timing for hover and focus states
  • • Non-essential information that doesn't block content access
