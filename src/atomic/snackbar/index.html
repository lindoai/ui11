Snackbar

Brief notification messages that appear temporarily to provide feedback about operations. Support for different message types and dismissible actions.

Notification Types

Different message types with semantic colors and appropriate icons for various contexts.

Usage Guidelines

  • • Use for brief messages about operation results or system status
  • • Auto-dismiss after 3-5 seconds unless user interaction is required
  • • Position consistently (usually bottom or top of screen)
  • • Limit to one snackbar visible at a time
  • • Include dismiss button for important messages

Accessibility Features

  • • ARIA live regions for screen reader announcements
  • • Semantic colors with sufficient contrast ratios
  • • Keyboard accessible dismiss buttons
  • • Clear, concise messaging for all user types
  • • Respects reduced motion preferences
