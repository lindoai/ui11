Select

Dropdown selection components with single and multi-select capabilities, search functionality, and organized option groups. Essential for forms and data selection interfaces.

Usage Guidelines

  • • Use single select for mutually exclusive choices with limited options
  • • Use multi-select when users need to choose multiple values from a list
  • • Group related options using optgroup for better organization
  • • Include search functionality for lists with many options (10+)
  • • Provide clear placeholder text to indicate the selection purpose

Accessibility Features

  • • Proper form labels and ARIA attributes for screen readers
  • • Keyboard navigation support (Tab, Arrow keys, Enter, Escape)
  • • High contrast focus indicators with proper ratios
  • • Option groups announced to assistive technologies
  • • Search functionality accessible via keyboard and screen readers
