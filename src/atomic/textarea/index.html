Textarea

Multi-line text input fields for longer content with resize functionality, validation states, and character counting. Essential for forms, comments, and extended text input.

Usage Guidelines

  • • Use textareas for multi-line content like comments, descriptions, or messages
  • • Set appropriate minimum rows (4-6) to provide adequate initial space
  • • Enable vertical resize (resize-y) to let users adjust height as needed
  • • Include character limits and counters for content with restrictions
  • • Provide clear error messages and validation feedback

Accessibility Features

  • • Proper form labels and ARIA attributes for screen readers
  • • Keyboard navigation support (Tab, Shift+Tab, Enter for new lines)
  • • High contrast focus indicators with proper ratios
  • • Clear validation messaging linked to textareas
  • • Character count announcements for assistive technologies
