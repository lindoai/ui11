Usage Guidelines

  • • Use Primary buttons for main actions (save, submit, continue)
  • • Use Secondary buttons for supporting actions (cancel, back)
  • • Use Ghost buttons for subtle actions (learn more, view details)
  • • Maintain consistent button sizes within the same context

Accessibility Features

  • • ARIA labels and roles for screen readers
  • • Keyboard navigation support (Tab, Enter, Space)
  • • Focus indicators with proper contrast ratios
  • • Disabled state clearly indicated
