Skeleton Loader

Content placeholders with subtle animations to indicate loading states and improve perceived performance.

Text Skeleton

Placeholder lines for text content with varying widths.

<!-- Heading Skeleton -->
<div class="h-8 bg-neutral-200 dark:bg-neutral-700 rounded-lg w-3/4 animate-pulse"></div>

<!-- Paragraph Skeleton -->
<div class="space-y-3">
  <div class="h-4 bg-neutral-200 dark:bg-neutral-700 rounded w-full animate-pulse"></div>

  <div class="h-4 bg-neutral-200 dark:bg-neutral-700 rounded w-4/5 animate-pulse"></div>
</div>

Usage Guidelines

  • • Use skeleton loaders for initial page loads and content that takes time to fetch
  • • Match skeleton structure to the actual content layout as closely as possible
  • • Keep animations subtle and non-distracting with proper timing
  • • Remove skeleton loaders as soon as real content is available
  • • Consider accessibility and provide appropriate loading announcements

Accessibility Features

  • • Reduced motion support for users with vestibular sensitivities
  • • Screen reader announcements for loading states
  • • High contrast placeholders that work in both light and dark modes
  • • ARIA live regions to announce when content has loaded
  • • Semantic structure that maintains page layout integrity
