Usage Guidelines

  • • Use external link icons for links that open in new tabs or windows
  • • Apply underlines to links within body text for better accessibility
  • • Use disabled styling for unavailable or inactive links
  • • Maintain consistent hover states across all link types
  • • Include proper ARIA attributes for current page indicators

Accessibility Features

  • • High contrast colors that meet WCAG accessibility standards
  • • Clear focus indicators for keyboard navigation
  • • Proper target and rel attributes for external links
  • • aria-current attribute for current page indicators
  • • Descriptive link text that makes sense out of context
