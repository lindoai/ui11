Usage Guidelines

  • • Use fluid containers for full-width sections and hero areas
  • • Use fixed containers for content with specific width requirements
  • • Use responsive containers for adaptive layouts across screen sizes
  • • Always include horizontal padding to prevent edge-to-edge content

Accessibility Features

  • • Maintains readable line lengths for optimal reading experience
  • • Responsive design ensures content accessibility on all devices
  • • Proper spacing prevents cramped layouts on mobile devices
  • • Centers content for balanced visual presentation
