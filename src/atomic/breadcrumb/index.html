Usage Guidelines

  • • Use consistent separators throughout your application
  • • Always show the current page as non-clickable text
  • • Implement truncation for long breadcrumb paths
  • • Consider mobile-responsive patterns for small screens
  • • Include proper ARIA labels for accessibility

Accessibility Features

  • • Semantic nav and ol elements for proper structure
  • • aria-label attributes for navigation context
  • • title attributes for truncated text
  • • Keyboard accessible interactive elements
  • • High contrast separators and text colors
