Usage Guidelines

  • • Use number badges for counts that require exact values (notifications, items)
  • • Use dot badges for simple status indicators and presence states
  • • Use status badges for categorical information and content labeling
  • • Keep badge text concise and meaningful (preferably single words)
  • • Position badges consistently near related content

Accessibility Features

  • • High contrast colors meet WCAG accessibility standards
  • • Semantic color usage with consistent meaning across contexts
  • • Screen reader friendly with descriptive text content
  • • Focus indicators for interactive badge elements
  • • Tooltip support for dot badges with title attributes
