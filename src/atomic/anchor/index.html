Usage Guidelines

  • • Add scroll-smooth class to html element for smooth scrolling
  • • Use scroll-margin-top for offset when you have fixed headers
  • • Implement active state tracking for long-form content navigation
  • • Ensure anchor targets have sufficient contrast and visibility

Accessibility Features

  • • Keyboard navigation with Tab and Enter keys
  • • Focus indicators that meet WCAG contrast requirements
  • • Skip links for main content navigation
  • • Proper heading hierarchy for screen readers
