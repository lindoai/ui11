Usage Guidelines

  • • Use announcement banners for general information and updates
  • • Use warning banners for important alerts and critical notifications
  • • Use promotion banners for marketing messages and special offers
  • • Always provide a way to dismiss non-critical banners
  • • Keep banner text concise and actionable

Accessibility Features

  • • High contrast colors meet WCAG accessibility standards
  • • Semantic color coding for different banner types
  • • Keyboard accessible dismiss buttons
  • • Screen reader compatible with proper ARIA labels
  • • Clear visual hierarchy with icons and typography
