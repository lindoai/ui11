Usage Guidelines

  • • Use consistent icon sizes throughout your interface (16px, 20px, 24px, 32px)
  • • Choose outline icons for general interface elements and filled for active states
  • • Use semantic colors to convey meaning (success, warning, error)
  • • Ensure icons have proper contrast ratios for accessibility
  • • Always include aria-label for icon-only buttons

Accessibility Features

  • • High contrast colors that meet WCAG accessibility standards
  • • Scalable vector format ensures crisp display at all sizes
  • • Proper aria-label attributes for screen reader compatibility
  • • Color inheritance from parent text color for consistency
  • • Semantic HTML structure with meaningful icon choices
