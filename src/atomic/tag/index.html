Usage Guidelines

  • • Use closable tags for user selections that can be removed (filters, chosen items)
  • • Apply semantic colors consistently across your application for better recognition
  • • Choose tag sizes based on content hierarchy and available space
  • • Keep tag text concise and descriptive (1-2 words when possible)
  • • Group related tags together and maintain consistent spacing

Accessibility Features

  • • High contrast color combinations for WCAG compliance
  • • Descriptive ARIA labels for close buttons with context
  • • Keyboard navigation support for interactive tags
  • • Focus indicators visible on all interactive elements
  • • Screen reader friendly with meaningful text content
