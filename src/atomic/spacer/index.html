Usage Guidelines

  • • Use the 8px-based spacing scale for consistent rhythm
  • • Prefer space-y-* and space-x-* for consistent element spacing
  • • Use responsive spacing to improve layouts across devices
  • • Combine margin and padding thoughtfully to avoid spacing conflicts
  • • Consider using gap utilities with flexbox and grid layouts

Accessibility Features

  • • Consistent spacing improves visual hierarchy and readability
  • • Responsive spacing ensures content accessibility on all devices
  • • Proper spacing prevents content from feeling cramped or overwhelming
  • • Adequate padding improves touch target accessibility on mobile devices
